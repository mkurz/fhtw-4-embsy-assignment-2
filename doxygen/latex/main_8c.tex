\+: Illustrates intertask synchronization with semaphores. Two producer tasks, in parallel, each increments a number of a global struct and saves the number in the same global struct inside an accompanying char array. Meanwhile, a consumer task divides the values by half (if they are odd) and updates the according char array as well. A fourth task outputs the values and the char array via the U\+A\+R\+T2. When not using semaphores the number values and their accompanying char arrays diverge. This can be tested by removing the \char`\"{}\+U\+S\+E\+\_\+\+S\+E\+M\char`\"{} define.

Meanwhile, a consumer task divides the values by half (if they are odd) and updates the according char array as well. A fourth task outputs the values and the char array via the U\+A\+R\+T2.
